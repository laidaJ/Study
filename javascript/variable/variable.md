# Variable

- 变量

- 常量
- 基本数据类型

- 数组
- 字符串

## 变量

```sh
# 声明变量
let num = 10
console.log(num)
$ 10
```

## 常量

```sh
# 声明常量
const uname = lesen
console.log(uname)
$ lesen
```

## 基本数据类型

- String
- Number

- Undefined
- Null

### 转换

```sh
# + 两边只要有一个字符串,就都会转换成字符串
# -*/等会把数据转换成数字类型
# 隐式转换
let num = + '123'
console.log(num)
$ 123
# 显示转换
let num = Number('123')
# 取整数
console.log(parseInt('12.3px'))
$ 12
# 取小数
console.log(parseFloat('12.3px'))
$ 12.3
```
